# the parsed list is passed to mlflow log_model as pip_requirements).

# transformers is pinned <5 so one file can serve training and serving

# Model Serving pods as of mid-2026 (4.13+ opencv builds bundle an OpenSSL
# that fails the pods' FIPS self-test, and vllm>=0.15 floors opencv at 4.13).
# The base model's architecture must be in this vLLM version's supported
# list — re-verify these pins before changing the base model.
